Confidentialité

Dernière mise à jour : 31 janvier 2026

Nous accordons une grande importance à la protection de vos données personnelles. Cette politique de confidentialité explique comment l'application Noctua collecte, utilise, partage et protège les informations lors de l'utilisation de notre application. Elle s'applique à tous les utilisateurs de Noctua, que vous résidiez en France, dans l'Union européenne, aux États-Unis ou dans tout autre pays.

1. Responsable du traitement des données

Le responsable du traitement des données est Noctua, dont l'email de contact est : contact@noctua-app.com

Nous sommes basés à Paris, France.

2. Données collectées

2.1 Données de profil (fournies par l'utilisateur)

Lors de votre inscription et de l'utilisation de l'application, nous pouvons collecter :

Prénom Non Personnalisation de l'expérience

Adresse e-mail Email OU téléphone requis Gestion du compte
Numéro de téléphone Email OU téléphone requis Gestion du compte

2.2 Données collectées automatiquement

  • Identifiant utilisateur anonyme (UUID)
  • Fuseau horaire et langue
  • Type d'appareil et système d'exploitation (iOS ou Android)
  • Version de l'application

2.3 Données d'utilisation et analytiques

Pour améliorer notre application, nous collectons des données d'utilisation anonymisées :

  • Ouverture de l'application et fréquence d'utilisation
  • Progression dans l'onboarding
  • Interactions avec les citations (vues, likes, partages)
  • Requêtes de recherche effectuées dans l'application
  • Parcours quotidiens complétés
  • Impressions publicitaires

2.4 Données de diagnostic

Via les SDK intégrés (Firebase Analytics, Google AdMob), les données suivantes peuvent être collectées automatiquement :

  • Données sur les pannes : rapports de crash, traces d'erreurs
  • Données de performance : temps de chargement, latence réseau, réactivité
  • Autres données de diagnostic : logs techniques

Ces données sont collectées de manière anonymisée et sont utilisées pour améliorer la stabilité et les performances de l'application.

2.5 Identifiants publicitaires

  • iOS (IDFA) : Uniquement si vous donnez votre consentement via la fenêtre App Tracking Transparency d'Apple
  • Android (Google Advertising ID) : Géré par Google Play Services

4. Partage des données avec des tiers

4.1 Supabase (Infrastructure et base de données)

Données partagées : Profil utilisateur, données d'utilisation, sessions quotidiennes.

Finalité : Stockage sécurisé des données, authentification, fonctionnement.

Localisation : UE et/ou États-Unis.

Politique de confidentialité de Supabase

4.2 Google AdMob (Publicité)

Données partagées : Identifiant publicitaire (si consenti), données d'interaction avec les publicités, informations sur l'appareil.

Finalité : Affichage de publicités, mesure de performance.

Politique de confidentialité de Google

4.3 Réseaux publicitaires partenaires (SKAdNetwork)

Pour mesurer l'efficacité des campagnes publicitaires sur iOS, nous travaillons avec des réseaux partenaires via le framework SKAdNetwork d'Apple. Ces partenaires reçoivent uniquement des données d'attribution agrégées et anonymisées.

4.4 Expo / React Native (Notifications push)

Données partagées : Token de notification push.

Finalité : Envoi des notifications quotidiennes.

Politique de confidentialité d'Expo

Nous ne vendons jamais vos données personnelles à des tiers.

5. Consentement et publicité

5.1 Consentement RGPD (Union européenne)

Si vous résidez dans l'EEE, au Royaume-Uni ou en Suisse, nous utilisons la plateforme Google UMP pour recueillir votre consentement. Vous pouvez modifier vos préférences à tout moment via Informations personnelles > Gérer mes préférences publicitaires.

5.2 App Tracking Transparency (iOS)

Sur iOS, nous demandons votre autorisation via la fenêtre système d'Apple avant d'accéder à votre IDFA. Vous pouvez refuser sans impact.

5.3 Droit de refus (CCPA - Californie)

Vous avez le droit de refuser le partage de vos informations personnelles à des fins publicitaires. Contactez-nous à contact@noctua-app.com pour exercer ce droit.

6. Durée de conservation des données

Type de données Durée de conservation
Données de profil Jusqu'à suppression du compte + 30 jours
Données d'utilisation (analytics) 24 mois puis anonymisation
Données de diagnostic 24 mois puis suppression automatique

Sessions quotidiennes 24 mois puis anonymisation
Données de consentement 3 ans (obligation légale)

7. Transfert international des données

Les transferts internationaux de données sont encadrés par les Clauses Contractuelles Types (CCT) et le Data Privacy Framework UE-États-Unis.

8. Sécurité des données

  • Chiffrement en transit (HTTPS/TLS)
  • Authentification sécurisée via tokens JWT
  • Accès restreint (principe du moindre privilège)
  • Surveillance et journalisation
  • Hébergement certifié SOC 2

10. Notifications push

Si vous activez les notifications push :

  • Vous recevrez une notification quotidienne à l'heure choisie.
  • Votre dernière activité sera enregistrée.
  • Vous pouvez désactiver les notifications à tout moment.

11. Stockage local sur votre appareil

L'application stocke localement sur votre appareil :

  • Préférences utilisateur
  • Profil (hors ligne)
  • État du parcours quotidien
  • Paramètres de notifications

Ces données ne sont pas transmises à des tiers, sauf dans le cadre de la synchronisation avec nos serveurs.

12. Protection des mineurs

L'application Noctua est destinée aux personnes âgées de 18 ans et plus. Nous ne collectons pas intentionnellement de données personnelles auprès de mineurs. Si vous pensez qu'un mineur nous a fourni des informations personnelles, veuillez nous contacter à contact@noctua-app.com.

13. Modifications de cette politique

En cas de modification de cette politique de confidentialité :

  • La date de mise à jour sera modifiée en haut de cette page.
  • Une notification sera envoyée dans l'application.
  • Votre consentement sera re-sollicité si nécessaire.

14. Réclamation auprès d'une autorité de contrôle

Si vous estimez que vos droits ne sont pas respectés, vous pouvez adresser une réclamation à :

  • France : CNIL (www.cnil.fr)
  • Autre pays de l'UE : l'autorité de protection des données de votre pays.

15. Contact

Pour toute question relative à cette politique de confidentialité, vous pouvez nous contacter par email : contact@noctua-app.com
